What this template is for

CSVLink is not limited to one document type. If your document has repeatable fields and your data is stored in Excel or CSV, you can build a custom layout and connect each field to a column.

Frequently asked questions

Can I build a template from scratch?
Yes. Use the CSVLink editor to create a custom layout and link text fields to spreadsheet columns.
Can I import a PDF design?
Yes. You can import a one-page PDF and place mapped fields on top of it.
Does CSVLink change my spreadsheet?
No. CSVLink uses your uploaded data to generate documents; it does not need to edit the original file.
